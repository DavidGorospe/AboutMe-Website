Java Farming Simulator Project

This project was a Java program of a Farming Simulator video game, where a user is able to grow and manage their crops and animals. I worked with 5 other members to develop this program. I was repsonsible for displaying the user's current farm progress/status.

C Friend Chat Project

This project was a C program of a Chat program, where a user is able to login to an account, add friends who have accounts, and chat with them in real-time. This program used socket libraries to establish a server and manage connections of users.

HR System Project

This project is a personal project that I started to practice my Clean Architecture skills and object-oriented programming. It is supposed to be a Human-Resource system that companies can use to manage their employees' information within the company like their pay, hours worked, and other information.
